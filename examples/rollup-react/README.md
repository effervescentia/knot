## Usage

### Setup

Install project dependencies

```sh
npm install
# or
yarn install
```

### Build

Run `rollup` and generate a browser-ready `bundle.js`

```sh
npm build
# or
yarn build
```

### Start

Start `http-server` to serve the built files and launch browser

```sh
npm start
# or
yarn start
```

### Test

Run ui regression tests with `cypress`

```sh
npm test
# or
yarn test
```
