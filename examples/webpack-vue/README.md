## Usage

### Setup

Install project dependencies

```sh
npm install
# or
yarn install
```

### Start

Start `webpack` and launch browser

```sh
npm start
# or
yarn start
```

### Test

Run ui regression tests with `cypress`

```sh
npm test
# or
yarn test
```

### Build

Compiled code will be placed in a `dist/` directory

```sh
npm build
# or
yarn build
```
